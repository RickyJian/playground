# 933. Number of Recent Calls

請實做 RecentCounter 類別，此用於計算一段時間內 request 的數量。

* Constructor()：初始化為 0 的計數器。
* ping(int t)：添加時間為 t（單位：毫秒） 的 request

請回傳在 3000 毫秒內所有發生的 request 數量。

> 每一次調用 `Ping()` t 都會大於前一次調用的值
