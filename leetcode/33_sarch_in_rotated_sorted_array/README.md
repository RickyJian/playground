# 33. Search in Rotated Sorted Array

有一個升冪的有序數值陣列(nums)，此陣列中不包含重複數值。

從數值陣列中隨機選定一個位置 k 做反轉(nums[k], nums[k+1], ...., nums[0], nums[1], ..., nums[k-1])，

請找出 target 位於反轉後數值陣列中的位置， 若 target 不存在於陣列中回傳 `-1`。

> 此陣列不為空
