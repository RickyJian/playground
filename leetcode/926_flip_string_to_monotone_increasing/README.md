# 926. Flip String to Monotone Increasing

給定一組由 0 和 1 組成的字串，請翻轉字串中元素的值(0 > 1 或是 1 > 0)，

將此字串滿足 monotone increasing 的特性，並找出最小翻轉次數。

> monotone increasing: 隨機從字串中切分兩個子集必須滿足左邊子集皆為 0 右邊子集皆為 1 的字串。
