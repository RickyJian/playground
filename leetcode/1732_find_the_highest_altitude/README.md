# 1732. Find the Highest Altitude

一位騎士正在進行一次公路旅行。這次旅行包含了 n + 1 個不同海拔高度的點。

騎士從海拔高度為 0 的第 0 點開始他的旅程。你會得到一個長度為 n 的整數陣列 gain，

其中 gain[i] 表示從第 i 點到第 i + 1 點之間的海拔高度增加量（對於所有 0 <= i < n）。

請回傳旅程中最高的海拔高度。
