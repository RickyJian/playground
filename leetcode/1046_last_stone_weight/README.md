# 1046. 最後一塊石頭的重量

## 問題描述

給定一個整數陣列 `stones`，其中 `stones[i]` 表示第 `i` 顆石頭的重量。

你要和石頭玩一個遊戲。每一回合都選出最重的兩顆石頭並互相碰撞。假設兩顆石頭重量分別為 `x` 與 `y`，且 `x <= y`，碰撞結果如下：

- 若 `x == y`，兩顆石頭都會被摧毀。
- 若 `x != y`，重量為 `x` 的石頭被摧毀，重量為 `y` 的石頭變成 `y - x`。

遊戲結束後，最多只會剩下一顆石頭。

請回傳最後剩下的石頭重量；若沒有石頭剩下，回傳 `0`。

### 範例 1

- 輸入：`stones = [2,7,4,1,8,1]`
- 輸出：`1`

### 範例 2

- 輸入：`stones = [1]`
- 輸出：`1`

## 限制條件

- `1 <= stones.length <= 30`
- `1 <= stones[i] <= 1000`
