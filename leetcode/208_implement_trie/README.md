# 208. Implement Trie (Prefix Tree)

實做 Trie 功能：

> 功能：
>
> Trie()：初始 Trie 結構 <br>
> Insert(String word)：新增 word 進 Trie <br>
> Search(String word)：查詢 word 是否存在於 Trie 中，若有回傳 True <br>
> StartsWith(String prefix)：查詢前綴字是否存在於 Trie 中，若有回傳 True
