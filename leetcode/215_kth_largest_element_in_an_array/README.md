# 215. 陣列中第 K 個最大元素

## 問題描述

給定一個整數陣列 `nums` 和一個整數 `k`，請回傳陣列中第 `k` 個最大的元素。

請注意，你需要找的是排序後陣列中第 `k` 個最大的元素，而不是第 `k` 個不同的元素。

### 範例 1

- 輸入：`nums = [3,2,1,5,6,4]`，`k = 2`
- 輸出：`5`

### 範例 2

- 輸入：`nums = [3,2,3,1,2,4,5,5,6]`，`k = 4`
- 輸出：`4`

## 限制條件

- `1 <= k <= nums.length <= 10^5`
- `-10^4 <= nums[i] <= 10^4`
