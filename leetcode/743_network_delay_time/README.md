# 743. Network Delay Time

有 n 個節點的網路圖，從一個節點到下一個節點皆有花費的時間(times)，

times[i] = (ui, vi, wi)：ui 原始節點，vi 連接節點，wi 花費時間

從某個節點 k 發出訊號，試問需要多久才能使所有節點都收到訊號，若此訊號無法到達所有節點請回傳 -1
