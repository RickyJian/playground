# 2099. 找出長度為 K 且總和最大的子序列

## 問題描述

給定一個整數陣列 `nums` 和一個整數 `k`，請從 `nums` 中找出長度為 `k` 且總和最大的子序列，以整數陣列的形式返回。

子序列（Subsequence）是指透過刪除某些元素（或不刪除），且不改變剩餘元素相對順序所得到的陣列。

**注意：** 可能存在多個有效答案，回傳任一皆可。

## 範例

**範例 1：**
```
輸入：nums = [2,1,3,3], k = 2
輸出：[3,3]
說明：子序列的最大總和為 3 + 3 = 6
```

**範例 2：**
```
輸入：nums = [-1,-2,3,4], k = 3
輸出：[-1,3,4]
說明：子序列的最大總和為 -1 + 3 + 4 = 6
```

**範例 3：**
```
輸入：nums = [3,4,3,3], k = 2
輸出：[3,4]
說明：子序列的最大總和為 3 + 4 = 7（[4,3] 也是有效答案）
```

## 限制條件

- `1 <= nums.length <= 1000`
- `-10^5 <= nums[i] <= 10^5`
- `1 <= k <= nums.length`
